# Laravel + Vue 表单验证九浅一深

Web 开发技术日新月异，一堆框架各领风骚，然而有一个课题却一直没变，那就是表单验证。

正所谓……你大爷永远是你大爷。

毕竟 TNT 仍未普及 :smile: ，手输表单仍然是人机进行交互的主要形式之一，如何友好而高效的进行表单验证，对于优化用户体验、提高网站安全性都很重要。
